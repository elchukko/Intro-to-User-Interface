CIS110 - Information Systems Concepts

Book: Computer Concepts 12th edition;
ISBN: 13-222-222-222-2222;
Prerequisite: Basic knowledge of Microsoft Windows. Competent keyboard skills.
Explore the fundamentals of information processing. Topics include: hardware, software, networking, the Internet, programming, and databases.

CIS125 - Intro to Dreamweaver


Book: Adobe Dreamweaver Revealed;
ISBN: 13-222-222-222-2222;
Prerequisite: Basic knowledge of file management.
Explore the fundamentals of Dreamweaver to create web pages.


CIS150 - Survey of Computing (IC3)


Book: IC3 3rd edition;
ISBN: 13-222-222-222-2222;
Prerequisite: none.
Learn basic concepts of word processing, spreadsheets, presentations, Internet, operating systems, and hardware using Internet and Computing Core Certification standards.
